# Dicas para Calouros – GitHub

## 1. Entenda o que é um repositório

O repositório é onde ficam todos os arquivos do projeto dentro do GitHub

## 2. Faça commits com frequência

Sempre que fizer mudanças no projeto, registre elas com um commit

## 3. Use mensagens claras nos commits

Escreva mensagens simples explicando o que foi alterado

## 4. Trabalhe com branches

Crie uma branch com seu nome para trabalhar sem alterar o projeto principal

## 5. Atualize antes de enviar mudanças

Antes de dar push, atualize o repositório para evitar conflitos

## 6. Organize bem os arquivos

Use nomes claros e mantenha os arquivos organizados

## 7. Use o GitHub para aprender

Explore outros projetos e veja como outras pessoas organizam seus códigos
